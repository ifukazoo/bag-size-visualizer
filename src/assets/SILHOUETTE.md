# シルエット画像アセット作成手順

Bag Size Visualizer で使う人物シルエット画像を**差し替える／作り直す**ときの手順書。

## アセット一覧

| ファイル | 内容 | 背景 |
|---|---|---|
| `silhouette.png` | 人物シルエット全体（下地） | 不透過で可 |
| `silhouette_hands.png` | バッグの手前に出す「手だけ」の画像（手以外は透明） | 透過必須 |

描画は3層で合成される：`① silhouette.png → ② バッグ画像 → ③ silhouette_hands.png`。
これにより「手でバッグを持っている」見え方になる（手レイヤーが常にバッグの最前面）。

## 鉄則

- **2枚は同じキャンバスサイズ・同じ位置**で書き出す。`silhouette_hands.png` は `silhouette.png` を複製してマスクしたものにする（別途AIで手だけ描かせると位置・色がズレるので NG）。
- `silhouette_hands.png` に残すのは「**バッグの手前に出すべき画素だけ**」（後ろに隠れる腕などは残さない）。
- 自動トリミング（不透明部分で切り詰める）はオフ。サイズが変わると合成位置がズレる。

## 手レイヤーの作り方（Photopea・無料・ブラウザ）

シルエットを AI 生成すると平面画像（レイヤーなし）になるため、生成画像から手を切り抜いて作る。

1. `File → Open` で `silhouette.png` を開く
2. レイヤーパネルで**背景レイヤーをダブルクリックして通常レイヤーに変換**（ロック解除）
   - ⚠️ これを忘れると消した部分が透明にならず背景色で塗られる（最重要）
3. なげなわツール（`L`）／多角形なげなわ／ペンツールで、**バッグの手前に出したい手の部分**を囲む
4. `Select → Inverse`（`Ctrl+Shift+I`）で選択を反転
5. `Delete`（Backspace）で手以外を消去 → 透明になる
6. `Ctrl+D` で選択解除
7. `File → Export as → PNG` で書き出し。**幅・高さは変更しない**（同寸のまま）→ `silhouette_hands.png`

元画像のコピーを部分的に消すだけなので、下地とピクセルが完全一致する。

## 差し替え時にコード側で確認すること

`src/hooks/useBagCanvas.ts` がサイズ換算・合成を行う。

- **実寸サイズ換算**：人物の頭頂〜足先のピクセルが身長に対応する。画像を差し替えたら、新画像での「頭頂Y・足先Y」を Preview 等で目視測定し、`useBagCanvas.ts` の頭足比率の定数（`BODY_RATIO` 等）を更新する。画像全高ではなく**頭足ピクセル**を身長基準にしないとバッグの実寸比率がずれる。
- 差し替え後は `cd frontend && npm run build` と、ブラウザでの描画確認（iPad Pro 11" Portrait: 834×1194）を行う。
